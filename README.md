# Quadruped-robot
Here you will find all the files required for this project

Download the 3D printer files from-
https://www.thingiverse.com/thing:2732957

# Parts required:
- ESP8266 board
- 16 Channel PWM servo motor driver
- 8 Servo motors (most likely SG-90)
- 3D printed body
- Batteries
- Jumper wires
